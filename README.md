## Música
Dá para colocar música, mas o navegador bloqueia autoplay.
Para usar música:
1. Suba um arquivo chamado musica.mp3 na raiz.
2. Abra o index.html.
3. Remova o comentário da parte "Música opcional".
4. O site mostrará o botão "Ativar som".
